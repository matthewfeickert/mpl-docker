matplotlib.axes.Axes.set_xticks¶

Axes.set_xticks(ticks, *, minor=False)¶

Set the xaxis' tick locations.

Parameters:
tickslist of floats

List of tick locations.

minorbool, default: False

If False, set the major ticks; if True, the minor ticks.
