=======================
Adding lines to figures
=======================

Adding lines to a figure without any axes.


.. code-block:: default


    import matplotlib.pyplot as plt
    import matplotlib.lines as lines


    fig = plt.figure()
    fig.add_artist(lines.Line2D([0, 1], [0, 1]))
    fig.add_artist(lines.Line2D([0, 1], [1, 0]))
    plt.show()
