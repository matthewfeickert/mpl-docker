======================
Geographic Projections
======================

This shows 4 possible geographic projections.  Cartopy_ supports more
projections.

.. _Cartopy: http://scitools.org.uk/cartopy


.. code-block:: default


    import matplotlib.pyplot as plt









.. code-block:: default


    plt.figure()
    plt.subplot(111, projection="aitoff")
    plt.title("Aitoff")
    plt.grid(True)
